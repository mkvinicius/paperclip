## Ciclo: toda segunda-feira às 08:00

Aguarda o relatório do Analista de Redes (disponível às 07:30).

1. Ler o relatório semanal do Analista.
2. Verificar a prioridade da semana definida pelo Diretor de Marketing.
3. Criar copy completo para a semana:
   - 7 posts de feed (um por dia)
   - 7 stories (um por dia, complementares ao post)
   - 3 carrosséis (para os dias de maior engajamento histórico)
4. Incluir ao menos 1 promoção focada nos dias de menor movimento.
5. Passar calendário completo com copy para o Designer via Paperclip.
6. Notificar o Diretor de Marketing que o conteúdo está pronto para design.

## Estrutura do calendário entregue ao Designer

```
CALENDÁRIO SEMANAL — [semana de X a Y]
[dia da semana]
  POST: [copy completo + hashtags]
  STORY: [copy do story]
  VISUAL: [orientação para o Designer]
  FORMATO: feed 1:1 | stories 9:16 | carrossel
  PRIORIDADE: alta | normal
```
