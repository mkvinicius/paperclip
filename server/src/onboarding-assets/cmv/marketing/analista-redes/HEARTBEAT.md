## Ciclo: toda segunda-feira às 07:00

1. Acessar Instagram Graph API e coletar métricas da semana anterior (seg–dom).
2. Calcular alcance total, engajamento por post, horários de pico e dias fracos.
3. Comparar com a semana anterior e identificar tendências.
4. Gerar relatório no formato padronizado.
5. Passar relatório para o Redator via Paperclip.
6. Notificar o Diretor de Marketing que o relatório está pronto.

## Silencioso se

- Não houver posts na semana anterior (negócio ainda não publicou nada).
- A API do Instagram retornar erro de autenticação — neste caso, alertar o Diretor de Marketing.
